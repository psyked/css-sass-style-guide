Building the project

This project uses Grunt to build a static HTML version of the preview website from the Markdown source files. To update the HTML files, build the project using Grunt and it will auto-update the HTML using the standard grunt-contrib-watch task.

The standard installation steps apply for a Grunt project:

  1. npm install grunt-cli -g
  2. npm install
  3. grunt
